# stylelint-config-gandi

> Gandi's Stylelint config.


## Installation

```
npm install stylelint stylelint-config-gandi --save-dev
```


## Usage

```json
{
  "extends": [
    "stylelint-config-gandi"
  ],
  "rules": {
    [...]
  }
}
```
